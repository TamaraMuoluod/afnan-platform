# AFNAN Platform 🌿
**Smart Agriculture & Professional Services Marketplace**

## 📌 Overview
Afnan is a specialized platform designed to bridge the gap between Agricultural Engineering graduates and the labor market in Al-Anbar, Iraq. It serves as a hub for smart farming, landscaping, and professional agricultural consultations.

##  Key Features
- **Professional Directory:** A verified list of agricultural experts and graduates in Anbar.
- **Smart Booking:** Facilitating service requests for home gardens, public parks, and government sectors.
- **AI Diagnostics:** An integrated AI assistant to diagnose plant diseases and recommend the right expert.
- **IoT Integration:** A solar-powered smart irrigation monitoring system using Arduino to support sustainability.

##  Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js & Express.js
- **Database:** MongoDB
- **Hardware:** Arduino & Soil Moisture Sensors.
